# CareerPath AI
AI Career Roadmap Generator using Flask + Groq + SQLite

Generates personalized screening tests and 4/8 week roadmaps based on skills & interests. Uses Groq LLaMA3 for AI questions. Includes progress tracking and auto resume builder.

## Stack
Backend: Flask, SQLite, Groq API  
Frontend: HTML, CSS, JS  
AI: Groq llama3-8b-8192

## Run Locally
```bash
cd careerpath
pip install -r requirements.txt
python app.py
